# BUG-005: ReleaseHealthCheckPage — Release todo'ları yüklenmiyor (yanlış query param)

## Sorun

Release Health Check sayfasında "Release Todos" bölümü hiç veri göstermiyor. Versiyon seçilse bile todos gelmiyor.

## Adımlar (Reproduce)

1. Login yap
2. Sol menüden "Release Health Check" sayfasına git
3. Bir ürün ve versiyon seç
4. "Release Todos" bölümünü gözlemle

## Beklenen Davranış

Seçilen versiyona ait todo'lar görünmeli.

## Gerçekleşen Davranış

Todo listesi boş. Backend 400 hatası dönüyor.

## Teknik Kök Neden

**Satır 79:**
```typescript
queryFn: () => apiClient.get(`/release-todos?productVersionId=${selectedVersion}`).then(r => r.data.data ?? r.data),
```

Backend'in beklediği query param `versionId`, ancak frontend `productVersionId` gönderiyor:
```typescript
// backend releaseTodos.routes.ts:
const { versionId } = req.query;
if (!versionId) throw new AppError(400, 'versionId gerekli');
```

## Önerilen Fix

```typescript
// L79 — productVersionId → versionId
queryFn: () => apiClient.get(`/release-todos?versionId=${selectedVersion}`).then(r => r.data.data ?? r.data),
```
