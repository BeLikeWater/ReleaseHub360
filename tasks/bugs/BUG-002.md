# BUG-002: ReleaseTodosPage — Ürün ve versiyon dropdown'ları boş geliyor

## Sorun

Release Todos sayfasında Product ve Version dropdown'ları boş, todo listesi gelmiyor.

## Adımlar (Reproduce)

1. Login yap
2. Sol menüden "Release Todos" sayfasına git
3. Product dropdown'ını aç

## Beklenen Davranış

Dropdown'da ürün listesi görünmeli, seçince versiyonlar gelmeli.

## Gerçekleşen Davranış

Dropdown boş.

## Teknik Kök Neden

BUG-001 ile aynı pattern. `r.data` yerine `r.data.data` kullanılmalı.

**Satır 56:**
```typescript
queryFn: () => api.get('/products').then((r) => r.data),
```

**Satır 61:**
```typescript
queryFn: () => api.get(`/product-versions?productId=${productId}`).then((r) => r.data),
```

## Önerilen Fix

```typescript
// L56
queryFn: () => api.get('/products').then((r) => r.data.data),

// L61
queryFn: () => api.get(`/product-versions?productId=${productId}`).then((r) => r.data.data),
```
