# BUG-020: CustomerDashboardPage — TransitionsSection tablosunda ürün adı/versiyon kolonu yok

## Sorun

`TransitionsSection` bileşeni, `GET /customer-version-transitions?customerId=...` ile
müşterinin **tüm ürünlerine** ait geçiş planlarını çekiyor. Ancak tablo sadece
`Ortam | Durum | Planlanan Tarih | Gerçekleşen Tarih` kolonlarını gösteriyor.

Bir müşteri birden fazla ürün geçişi planlamışsa, tabloda hangi geçişin hangi ürüne ait
olduğu **ayırt edilemiyor**.

## Etkilenen Kod

```tsx
// packages/frontend/src/pages/CustomerDashboardPage.tsx ~line 211-235
<TableHead>
  <TableRow sx={{ bgcolor: 'action.hover' }}>
    <TableCell sx={{ fontWeight: 700 }}>Ortam</TableCell>
    <TableCell sx={{ fontWeight: 700 }}>Durum</TableCell>
    <TableCell sx={{ fontWeight: 700 }}>Planlanan Tarih</TableCell>
    <TableCell sx={{ fontWeight: 700 }}>Gerçekleşen Tarih</TableCell>
    {/* ❌ Ürün adı / versiyon kolonu yok */}
  </TableRow>
</TableHead>
```

## Adımlar (Reproduce)

1. Bir müşteri için 2 farklı ürünün geçişini planla
2. Customer Dashboard'u aç, sayfanın altındaki "Geçiş Planları" tablosuna bak
3. İki farklı ürünün geçişlerinin aynı tablo satırlarında hangi ürüne ait olduğu belli olmadığını gör

## Beklenen Davranış

Tablo `Ürün | Hedef Versiyon | Ortam | Durum | Planlanan Tarih | Gerçekleşen Tarih` kolonlarını
içermeli.

## Gerçekleşen Davranış

Ürün adı ve hedef versiyon gösterilmiyor. Birden fazla ürün geçişi olan müşterilerde
tablo anlamsız hale geliyor.

## Teknik Kök Neden

`Transition` interface'i ve `GET /customer-version-transitions` backend response'u:
```json
{
  "id": "...",
  "environment": "PROD",
  "status": "PLANNED",
  "plannedDate": "...",
  "toVersionId": "...",
  ...
}
```

`GET /` endpoint'i `cvtInclude` ile `toVersion.product.name` ve `toVersion.version` döndürüyor
(backend'de `include` mevcut), ancak frontend `Transition` interface'inde bu alanlar tanımlı değil
ve tabloda kullanılmıyor.

## Önerilen Düzeltme

### Frontend: `Transition` interface güncellenmeli

```tsx
interface Transition {
  id: string;
  // ... mevcut alanlar ...
  toVersion?: {
    id: string;
    version: string;
    product: { id: string; name: string };
  };
}
```

### Frontend: Tablo başlığına kolon eklenmeli

```tsx
<TableCell sx={{ fontWeight: 700 }}>Ürün</TableCell>
<TableCell sx={{ fontWeight: 700 }}>Hedef Versiyon</TableCell>
<TableCell sx={{ fontWeight: 700 }}>Ortam</TableCell>
// ...
```

### Frontend: Tablo satırında gösterilmeli

```tsx
<TableCell>
  <Typography variant="caption">{t.toVersion?.product?.name ?? '—'}</Typography>
</TableCell>
<TableCell>
  <Chip size="small" label={t.toVersion?.version ?? '—'} sx={{ fontFamily: 'monospace' }} />
</TableCell>
```
