# BUG-004: CodeSyncPage — "Sync Başlat" butonu 404 hatası veriyor

## Sorun

Code Sync sayfasında wizard doldurulduktan sonra "Sync Başlat" butonuna tıklanınca istek 404 döndürüyor. Sync başlamıyor.

## Adımlar (Reproduce)

1. Login yap
2. Sol menüden "Code Sync" sayfasına git
3. Wizard form'unu doldur (customer, target branch vb.)
4. "Sync Başlat" butonuna tıkla

## Beklenen Davranış

MCP Server üzerinden sync işlemi başlamalı.

## Gerçekleşen Davranış

404 hatası. Sync başlamıyor.

## Teknik Kök Neden

Frontend `line 56`:
```typescript
mutationFn: (data: typeof wizardForm) => apiClient.post('/code-sync/start', data),
```

Ancak backend'de `/code-sync/start` endpoint'i yok. Mevcut endpoint:
```typescript
router.post('/execute', async (req, res, next) => { ... })
// → /api/code-sync/execute
```

## Önerilen Fix

```typescript
// L56 — /start → /execute
mutationFn: (data: typeof wizardForm) => apiClient.post('/code-sync/execute', data),
```
