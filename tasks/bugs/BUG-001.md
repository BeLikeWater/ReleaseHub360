# BUG-001: ReleaseNotesPage — Ürün ve versiyon dropdown'ları boş geliyor

## Sorun

Release Notes sayfasında Product ve Version dropdown'ları seçimde boş kalıyor. Ekran açılınca ürün seçilemiyor, versiyon seçilemiyor dolayısıyla release note listesi hiç gelmiyor.

## Adımlar (Reproduce)

1. Login yap
2. Sol menüden "Release Notes" sayfasına git
3. Product dropdown'ını aç

## Beklenen Davranış

Dropdown'da "Customer Portal", "ERP Core" gibi ürünler görünmeli.

## Gerçekleşen Davranış

Dropdown boş. Seçenek yok.

## Teknik Kök Neden

`products` ve `product-versions` query'lerinde `r.data` dönüyor. Ancak `apiClient` axios instance'ı ve backend `{ data: [...] }` formatında döndürüyor. Dolayısıyla `r.data = { data: [...] }` ve bu bir obje, dizi değil. `versions.find()` ve `products.map()` boş array'e karşı çalışıyor.

**Satır 68:**
```typescript
queryFn: () => api.get('/products').then((r) => r.data),
// r.data = { data: [{id:..., name:...}] }  ← obje
// Olması gereken: r.data.data
```

**Satır 73:**
```typescript
queryFn: () => api.get(`/product-versions?productId=${productId}`).then((r) => r.data),
// Aynı sorun
```

## Önerilen Fix

```typescript
// L68
queryFn: () => api.get('/products').then((r) => r.data.data),

// L73
queryFn: () => api.get(`/product-versions?productId=${productId}`).then((r) => r.data.data),
```
