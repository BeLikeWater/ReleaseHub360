# BUG-007: ReleaseHealthCheckPage — Versiyon seçince sayfa crash oluyor (prs.filter is not a function)

## Sorun

Kullanıcı Release Health Check sayfasında bir ürün ve versiyon seçtiğinde ekran tamamen çöküyor.
Konsol hatası: `TypeError: prs.filter is not a function`

## Adımlar (Reproduce)

1. Login yap
2. Sol menüden "Release Health Check" sayfasına git
3. "Ürün" dropdown'ından herhangi bir ürün seç
4. "Versiyon" dropdown'ından bir versiyon seç
5. Sayfa hata ekranı gösterir (crash)

## Beklenen Davranış

Versiyon seçildikten sonra sağlık skoru, PR listesi, pipeline durumu ve todo tabloları görüntülenmeli.

## Gerçekleşen Davranış

Sayfa tamamen çöküyor. `TypeError: prs.filter is not a function` hatası fırlatılıyor.

## Teknik Kök Neden

`/api/tfs/pull-requests` endpoint'i şu şekilde yanıt döndürüyor:
```json
{ "data": { "value": [...], "count": 101 } }
```

Ancak `ReleaseHealthCheckPage.tsx` satır ~82'deki queryFn:
```typescript
queryFn: () => apiClient.get('/tfs/pull-requests').then(r => r.data.data ?? r.data),
```
Bu ifade `r.data.data` = `{ value: [...], count: 101 }` şeklinde bir **obje** döndürüyor.
Default değer `= []` sadece `undefined` / `null` için geçerli; sorgu resolve olunca bu obje `prs` değişkenine atanıyor.

Ardından render aşamasında:
```typescript
const openPRs = prs.filter(p => p.status === 'active' || p.status === 'open');
```
`.filter()` bir Array metodu olduğundan, bir obje üzerinde çağrılınca `TypeError` fırlatıyor ve sayfa crash oluyor.

Karşılaştırma: Pipeline query aynı durumu **doğru** şekilde handle ediyor (satır ~92):
```typescript
queryFn: () => apiClient.get('/tfs/pipelines').then(r => {
  const d = r.data.data ?? r.data;
  return d.value ?? (Array.isArray(d) ? d : []);  // ← .value çıkarımı yapıyor
}),
```
PR query aynı pattern'ı eksik uyguluyor.

## Önerilen Fix

`packages/frontend/src/pages/ReleaseHealthCheckPage.tsx` satır ~82:

```typescript
// ÖNCE (hatalı):
queryFn: () => apiClient.get('/tfs/pull-requests').then(r => r.data.data ?? r.data),

// SONRA (doğru):
queryFn: () => apiClient.get('/tfs/pull-requests').then(r => {
  const d = r.data.data ?? r.data;
  return d.value ?? (Array.isArray(d) ? d : []);
}),
```
