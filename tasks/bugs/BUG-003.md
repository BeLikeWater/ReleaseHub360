# BUG-003: CustomerManagementPage — Müşteri listesi ve mapping dropdown'ları boş

## Sorun

Customer Management sayfasında müşteri listesi boş geliyor. "Yeni Müşteri" dışında hiçbir müşteri görünmüyor. Ayrıca Customer-Product Mapping tab'ında tüm dropdown'lar (customers, products, versions) boş.

## Adımlar (Reproduce)

1. Login yap
2. Sol menüden "Customer Management" sayfasına git
3. Müşteri listesini gözlemle

## Beklenen Davranış

3 müşteri görünmeli (seed datası var: Acme Corp, TechStart, GlobalBank).

## Gerçekleşen Davranış

Liste boş geliyor.

## Teknik Kök Neden

Birden fazla queryFn'de `r.data` kullanılmış, `r.data.data` olmalı:

```typescript
// L111 — customers listesi
queryFn: () => api.get('/customers').then((r) => r.data),

// L351 — customer-product-mappings listesi
queryFn: () => api.get('/customer-product-mappings').then((r) => r.data),

// L356 — mapping form'unda customers dropdown
queryFn: () => api.get('/customers').then((r) => r.data),

// L361 — mapping form'unda products dropdown
queryFn: () => api.get('/products').then((r) => r.data),

// L366 — mapping form'unda versions dropdown
queryFn: () => api.get(`/product-versions?productId=${mapForm.productId}`).then((r) => r.data),
```

## Önerilen Fix

Yukarıdaki 5 satırda `.then((r) => r.data)` → `.then((r) => r.data.data)` olarak değiştir.
